# Ubuntu Innovations (Pty) Ltd – IT Support Capstone Project

## 📌 Project Overview

This capstone project was completed as part of the **Google IT Support Professional Certificate** under the **BSG Tech Career Accelerator Program**.

The project simulates a real-world business scenario where a Junior IT Support Specialist is contracted to design and document a complete IT infrastructure solution for **Ubuntu Innovations (Pty) Ltd**, a fictional South African technology startup based in Cape Town.

The goal of this project is to demonstrate practical IT support, networking, system administration, cybersecurity, troubleshooting, and professional documentation skills.

---

# 🏢 Company Background

Ubuntu Innovations (Pty) Ltd is a rapidly growing technology startup with **25 employees** across multiple departments:

| Department           | Employees |
| -------------------- | --------- |
| Executive Management | 3         |
| Finance              | 4         |
| Human Resources      | 3         |
| Sales & Marketing    | 7         |
| Software Development | 8         |
| Total Employees      | 25        |

The company is relocating to a new office and requires:

* Secure wired and wireless networking
* Shared storage and printers
* User access control
* Backup and disaster recovery
* Cybersecurity implementation
* IT documentation and troubleshooting procedures

# 🌐 Network Infrastructure Overview

The proposed network infrastructure includes:

* ISP Fibre Connection
* Firewall Router
* 24-Port Managed Switch
* Wireless Access Points
* NAS Backup Storage
* Network Printers
* Departmental VLAN Segmentation

The infrastructure was designed to ensure:

* Scalability
* Security
* Reliability
* Efficient network management
